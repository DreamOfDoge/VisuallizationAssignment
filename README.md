# VisuallizationAssignment
可视化大作业项目展示


创建者：任俊毅

我基于已知的理论进行了一次宇宙建模，模拟在数十亿年尺度下生命是如何在宇宙中传播的。当然这个模型有很多不足和错误，并不能反应宇宙的真实情况，毕竟物理学是在不断发展变化的，我们不知道的还是比知道的多。



#模型公设
这是模型基于的理论基础以及特性
1，信息传播最高速度为光速。
这会导致在宇宙过于广阔的空间中，信息传播具有严重的滞后性。因此相距数千数万光年的文明在交流上必然存在严重的隔断，即使是母文明和她的殖民地。因此子文明必须为自己做主，这将导致独立。
2，宇宙的熵始终在增加。
现实中的宇宙有没有寿命还不能确定，而模拟宇宙的寿命是无限的。那么熵增将会导致资源有限。在无限的时间中，宇宙中所有的分子最终会具有同样的能量，也就是“热寂”。这说明宇宙的资源是有限的。虽然更高级的技术可以推迟热寂，但宇宙中可利用的能量会越来越少，最终用无可用。
3，模拟宇宙不会膨胀

#文明阶段
根据生命的生存能力，为生命划分出5个阶段
1：演化阶段。存在生命，但不存在文明
2：初级阶段。出现了智慧，但无星际航行能力
3：一级阶段。观测并航行到其他星系，但无永居能力
4：二级阶段。永居其他星系。抵抗部分星际灾害
5：三级阶段。摧毁其他星系。抵抗所有星际灾害

#程序过程
宇宙是1000*1000的画布。每个像素点代表了一个星系。每次模拟重启都会随机布置物质
每0.1秒，相当于宇宙过了100万年，进行一次判定。判定内容如下：
1，每一块物质进行随机判定是否会产生文明，这个概率很低很低。
2，文明是否会进入下一个阶段。
3，每个阶段的文明的行动：发现观测范围里的其他星系；殖民其他星系；
4，相遇的两个不同文明，如果文明等级相差2及以上。更高级的文明将会吞并低级文明。
5，文明用0到1的数字来衡量敌对性。数字越高说明和平的意愿越低。相遇的两个不同文明将会根据好战程度，随机进入和平或战争状态。高级文明对低级文明的战争必定胜利。
6，有随机因素导致文明倒退或灭亡：初级和一级阶段，文明有低概率自我消亡，回退到演化阶段。发生星际灾害，文明回退到演化阶段或彻底消亡。
7，文明有观测范围。文明等级越高，观测范围越大。发现观测范围内星系的概率越高
8，一块物质的资源是有限的，根据文明人口减少资源
9，初级阶段的文明不会消耗资源
10，如果一块物质的资源消耗殆尽，如果该上的文明没有星际殖民能力，就会灭亡。否则将会迁移到观测范围里的另一个星系。如果该星系上有其他文明，则会爆发文明存亡之战
